ThreeJS Showcase

A showcase into the capabilities of threejs. Created by Logan Trudel & Zachary Carmean

Project Overview

The Problem:


When completinga survey for the course on what features people wanted to present on. It was found that there was not a high number of our peers that knew about ThreeJS.

My Role:


User research, testing, prototyping, and visual design.

Team:


Zachary Carmean ( modeling/web developlent ), Logan Trudel ( UXUI/Web developlemt )

The Solution:

We sought to create a small demostration on some of the capabilities that a developer/user could utilize when creating with ThreeJS.

Tools:

Visual Studio Code, JavaScript, html, css

Process

Brainstorming


We decided on creating a couple ideas on how we wanted our project to turn out.


First, we had to figure out what parts of ThreeJS we wanted to show off. For that, we decided to showcase how easy it was to create a scene, import meshes, create lights, and import blender models into your projects/webpages using the GLTFLoader utility.


After we figured out a basic idead of what we wanted to show, we created a basic wireframe that would give us an achievable goal to set towards when creatign the webpage.


After creating the wireframe we skipped right to creating our first prototype. Our presenation date on the topic was closing in so we had to expidite the creation process quite a bit.
